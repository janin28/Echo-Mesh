# EchoMesh Client Dashboard Guide

## Overview
EchoMesh is a decentralized bandwidth-sharing platform. This dashboard allows you to monitor your node's performance, track earnings, and manage sessions in real-time.

## Key Features
- **Real-time Metrics**: Monitor ingress/egress rates, latency, and system resource usage.
- **Health Monitoring**: Track node connectivity, proxy status, and policy violations.
- **Payout Settlement**: Settle sessions using a transparent formula based on data transferred and node reputation.
- **Solana Integration**: Connect your wallet to receive payouts on the Solana Devnet.

## Getting Started
1. **Configure Your Node**: Go to the Configuration page to set your bandwidth caps and region preferences.
2. **Start the Daemon**: Click "Start Daemon" on the Overview page to begin sharing bandwidth.
3. **Monitor Performance**: Use the charts to track throughput and health status.
4. **Settle Earnings**: Visit the Sessions page to settle completed sessions and view your payout history.

## Performance Targets
- **CPU**: < 5% typical usage.
- **RAM**: < 200MB steady-state.
- **Latency**: < 250ms p95 regional target.
